# Matrix Operation

This repo contains the code for matrix operation

## Addition

```fortran
A = [1,2,3;4,5,6]
B = [1,2,3;4,5,6]

C = A + B
```

## Subtraction

```fortran
A = [1,2,3;4,5,6]
B = [1,2,3;4,5,6]

C = A - B
```

## Multiplication

```fortran
A = [1,2,3;4,5,6]
B = [1,2,3;4,5,6]

C = A * B
```
